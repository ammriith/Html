Three types of CSS style sheets

CSS (Cascading Style Sheet) is used to style and layout web pages, contrilling aspects like colors, ont and spacing

Inline Css

  • These are defined directly in the HTML elements using the style attributes

Internal Css

  • These are defined within a <style> tag in the <head> section of HTML document.

External Css

  • These are defined in a seperate CSS file, which is linked to the HTML document using a <link> tag in the <head>
